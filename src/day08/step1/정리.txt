- 타입이란: 데이터의 종류. 변수에 저장할 값의 형태를 정의
- 타입을 언제 사용해야 되는지: 데이터의 정확성과 안정성을 보장하기 위해 사용

- 클래스란?: 객체를 생성하기 위한 청사진(설계도). 객체의 상태를 나타내는 속성(멤버 변수)과 행동을 나타내는 메서드(함수)를 정의
- 클래스를 언제 사용해야 되는지: 복잡한 데이터를 구조화하고 재사용성을 높이기 위해 사용

- 객체란 무엇인지: 클래스의 인스턴스(instance). 클래스에 정의된 구조와 행동을 가지며, 실제 데이터를 담고 있음
- 객체를 언제 사용해야 되는지: 실세계의 개념을 프로그래밍에 반영하고, 클래스에서 정의한 구조와 행동을 실제로 사용할 때 필요

- 멤버변수란 무엇인지: 클래스 내부에 정의된 변수로, 객체의 상태를 나타냄
- 멤버변수는 언제 사용해야 되는지: 객체의 고유한 상태를 저장하고 관리할 때 사용

- 생성자란 무엇인지: 클래스의 인스턴스를 초기화하는 특별한 메서드. 객체가 생성될 때 자동으로 호출되어 멤버변수를 초기화하거나 필요한 초기 설정을 수행
- 생성자는 언제 사용해야 되는지: 생성자는 객체가 생성될 때 초기화가 필요할 때 사용

- 메소드란/함수란?: 클래스 내부에 정의된 함수로, 객체의 행동을 정의
- 메소드/함수는 언제 사용해야 되는지: 코드를 재사용 가능하게 만들고, 프로그램의 구조를 체계적으로 관리하기 위해 사용

- 매개변수란?: 함수나 메서드에 입력으로 전달되는 값을 받기 위해 사용하는 변수
- 매개변수를 언제 사용해야 되는지: 함수나 메서드가 외부로부터 입력을 받아 처리해야 할 때 사용

- 반환타입이란?: 함수나 메서드가 실행을 마친 후 반환하는 값의 타입
- 반환/리턴이란?: 함수나 메서드가 결과값을 호출한 곳으로 돌려주는 행위
- 반환/리턴은 언제 사용해야 하는지?: 함수나 메서드가 작업을 완료하고 결과를 호출한 곳으로 돌려줄 때
