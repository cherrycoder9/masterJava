상속
    - 부모로부터 자식이나 손자까지 물려주는 행위를 의미
    - 상위클래스가 하위클래스의 개념을 확장, 세분화 하는 의미

                        - 고래
            - 포유류     - 박쥐
    - 동물
            - 조류       - 참새
                        - 닭

            * 조류는 동물이다, 고래는 포유류다, 고래는 동물이다 O
        - 이유
            1. 계층구조파악
            2. 유지 보수
            3. 재사용성
            4. 다형성
        - 상속 선언하는 방법
            class 부모클래스명{}
            class 자식클래스명 extends 부모클래스{}
        - 자식 클래스는 부모클래스의 멤버변수와 메소드를 사용할 수 있다
            단) 부모클래스는 자식클래스의 멤버변수, 메소드를 사용할 수 없다.

        - 타입변환
            - 자식타입 -> 부모타입으로 타입변환 가능
            - 부모타입 -> 자식타입으로 타입변환이 불가능
                단) (상-하 관계) 본래의 객체가 자식타입에서 부모타입으로 변환된 객체이면 가능
                    - 동물(탄생), 동물 ---> 닭         (불가능)
                    - 닭(탄생), 닭 ---> 동물 ---> 닭   (가능)
                단) (좌-우 관계) 형제관계는 불가능하다
                    - 닭(탄생), 닭 ---> 참새           (불가능)

        - 객체의 타입 확인 (유효성검사/오류방지)
            객체명 instanceof 타입
            -- 해당 객체가 검사 타입에 포함하는지 확인, 포함하면 true, 아니면 false
