# 제품 관련 순서도

1. 사용자(등록) 제품코드, 제품명, 제품가격 입력 받는다.
    - Scanner 객체 이용한 입력
    - 해당 객체의 next(), nextInt() 함수를 이용한 각 자료형/타입 받는다.
2. 로직(컴퓨터)는 입력받은 데이터를 각 변수에 저장한다.
    - 자료형 선택
3. 종이박스(배열)내 빈 공간이 있는 체크/확인
    - for 문 이용한 배열(저장목록) 순회 하기
    - 배열 내 빈공간 = null
4. 제품코드, 제품명, 제품가격 이 묶음 => 클래스/객체
    - 객체 만들기 위해 먼저 클래스 생성
    - 클래스 기반으로 new 연산자를 이용한 객체 생성
    - 찾은 배열의 인덱스에 객체 저장
5. 빈 공간 체크 종료, 찾았을때/못찾을때 식별 변수
    - 저장 성공과 실패했다 증거, 변수 활용
    - 성공했으면 뒤를 순회 종료
6. 결과 출력
    - 가서 증거에 따라 결과 안내
    - if, print
